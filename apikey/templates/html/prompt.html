API Endpoints

Python example for chat endpoint:

r = requests.post("https://professorparakeet.com/api/chat", headers={"Authorization": "Bearer xxx"}, json={'prompt': str, 'model' : str, 'n': int, 'top_k': float, 'top_p': float, 'best_of': int, 'max_tokens': int, 'frequency_penalty': float, 'presense_penalty': float, 'temperature': float, 'beam': bool, 'early_stopping': bool, 'length_penalty': float })
print(r.json())

cURL example for text completion endpoint:

curl "https://professorparakeet.com/api/completion" --header 'Content-Type: application/json' --request POST -H "Authorization: Bearer xxx" --data-binary '{"prompt": str, "n": int, "model" : str, "top_k": float, "top_p": float, "best_of": int, "max_tokens": int, "frequency_penalty": float, "presense_penalty": float, "temperature": float, "beam': bool, "early_stopping": bool, "length_penalty": float }'
